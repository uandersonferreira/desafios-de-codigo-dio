Conhecendo as Tags HTML para Títulos <h1> a <h6>

As tags <h1> a <h6> são usadas para definir títulos HTML.

<h1> define o título mais importante. <h6> define o título menos importante.

Nota: Use apenas um <h1> por página - deve representar o título/assunto principal de toda a página. Além disso, não pule os níveis de título - comece com <h1>, depois use <h2> e assim por diante.

Exemplo

Os seis títulos HTML diferentes: 

            
                
This is heading 1

                
This is heading 2

                
This is heading 3

                
This is heading 4

                
This is heading 5

                
This is heading 6

Criando Parágrafos <p>

A <p> tag define um parágrafo.

Os navegadores adicionam automaticamente uma única linha em branco antes e depois de cada <p> elemento.

Dica: Use CSS para estilizar parágrafos .

Exemplo
